# Assembly Code Repository

This project was developed during my Erasmus exchange program in Germany as part of the Computer Architecture course.

A collection of 30 functional assembly programs for **MIPS** and **RISC-V** architectures.

## Repository Structure

```
AssemblyProjects/
├── mips/                    # 15 MIPS programs (MARS Simulator)
├── riscv/                   # 15 RISC-V programs (RARS/Venus)
└── README.md
```
